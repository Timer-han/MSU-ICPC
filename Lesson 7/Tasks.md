# A. Ревакцинация

ограничение по времени на тест1 s.

ограничение по памяти на тест512 MB

вывод standard output

Врачи создали аж целых m вакцин от коронавируса! Теперь перед Володей стоит сложная задача: решить, какими вакцинами уколоться. Он решил, что поставит себе n доз вакцин, нужно только понять, какие именно лучше ставить. Известно, что первый укол i-той вакцины повышает иммунитет на ai, а каждый последующий укол этой же вакциной — на bi. Помогите Володе определить, какого максимального уровня иммунитета можно достичь?



Входные данные

В первой строке задано одно целое число t (1≤t≤10000) — количество наборов входных данных. Затем следует описание t наборов входных данных, каждый из которых задан следующим образом.



В первой строке набора записаны два целых числа n и m (1≤n≤109, 1≤m≤100000) — требуемое количество уколов и количество доступных видов вакцин.



Каждая из следующих m строк описывает очередную вакцину и содержит два целых числа ai и bi (0≤ai,bi≤109) — увеличение уровня иммунитета от первого укола i-той вакциной и увеличение уровня иммунитета от каждого последующего укола этим типом вакцины.



Наборы разделены одной пустой строкой.



Гарантируется, что сумма m по всем наборам входных данных не превосходит 100000.



Выходные данные

Для каждого набора входных данных в единственной строке выведите одно число — максимальный уровень иммунитета, который может получить Володя от n уколов.



Пример

входные данные

4 3

# C. Subtraction sort

time limit per test2 s.

memory limit per test256 MB

inputstandard input

outputstandard output

You are given an integer array a of size n.



You can perform operations on the array a. In each operation, you may choose three indices x,y,z:1≤x<y<z≤n and set ax=ay−az. After any operation you should hold ∀i:1≤i≤n:|ai|<1018.



Determine if you can turn array a to sorted in non-decreasing order, using no more than n operations.



Input

The first line contains integer t (1≤t≤104)– number of test cases.



The first line of each test case contains integer n (3≤n≤2⋅105) – size of array a.



The second line of each test case contains n integers ai (−109≤ai≤109) – array a itself.



It is guaranteed that sum of n across all test cases doesn't exceed 2⋅105.



Output

For each test case print −1 if there is no solution.



If solution exists, print number of operations needed m (0≤m≤n). On i-th of the next m lines print xiyizi – indices of i-th operation.



If there are multiple solutions, you can output any. Note that you don't have to minimize the number of operations.



Example

input

3

5

5 -4 2 -1 2

3

4 3 2

3

-3 -2 -1

output

2

1 2 3

3 4 5

-1

0

# D. ППСП

ограничение по времени на тест 1 s.

ограничение по памяти на тест 256 MB

ввод standard input

вывод standard output

Назовем последовательность из символов «(» и «)» правильной скобочной последовательностью (ПСП), если в нее можно вставить символы «1» и «+» и получить корректное математическое выражение. Например, скобочные последовательности «()()», «(())» — правильные (полученные выражения: «(1+1)+(1+1)», «((1+1)+1)»), а «)(» и «)» — нет. Пустая строка является правильной скобочной последовательностью по определению.



Назовем ПСП простой (ППСП), если ее нельзя разбить в непересекающееся объединение ПСП.



Вам дана строка, состоящая из символов «(», «)» и «?». Замените знаки вопроса на скобки таким образом, чтобы получить ППСП, или скажите, что так сделать нельзя. Если существует несколько решений, вы можете найти любое.



Входные данные

В первой строке записано одно целое число |s| (1≤|s|≤3⋅105) — длина строки.



Вторая строка содержит строку s, которая содержит только символы «(», «)», «?».



Выходные данные

Выведите одну строку, содержащую ответ на задачу.



Если возможных решений несколько, вы можете вывести любое.



Если решения не существует, выведите единственную строку, содержащую «:(» (без кавычек).



Примеры

входные данные

6

(?????

выходные данные

(()())

входные данные

10

(???(???(?

выходные данные

:(

Примечание

Можно доказать, что не существует ответа на второй тест  из примера, поэтому требуется вывести «:(».



# E. Грузная жизнь

ограничение по времени на тест 2 s.

ограничение по памяти на тест 256 MB

ввод standard input

вывод standard output

Грузчик Вася стоит на бесконечной прямой в точке 0. За один шаг он может переместиться на 1 влево или вправо. Также на прямой находятся n одинаковых коробок с грузами и m домов с покупателями, которые заказали себе на дом эти коробки.



Все бы ничего, но грузы слишком большие и тяжелые, чтобы их поднять, а Вася слишком одномерный, чтобы их обойти, поэтому единственное, что он может — толкать эти грузы. Более формально, когда Вася двигается вправо, то, если в его новой позиции стоит коробка, то она сдвигается на 1 вправо. При этом, если на новой позиции коробки стоит еще одна коробка, то и она сдвинется направо, и так далее.



Задача Васи — доставить как можно больше грузов до домов покупателей (покупатели заберут коробки в конце, после того, как Вася закончит свою работу).



Входные данные

В первой строке записано одно целое число t (1≤t≤1000) — количество наборов входных данных.



Затем следует описание t наборов входных данных.



В первой строке каждого набора входных данных записаны два целых числа n и m (1≤n,m≤2⋅105) — количество коробок и количество домов покупателей, соответственно.



Во второй строке каждого набора входных данных записаны n различных целых чисел в возрастающем порядке a1,a2,…,an (−109≤a1<a2<⋯<an≤109; ai≠0) — начальные позиции коробок.



В третьей строке каждого набора входных данных записаны m различных целых чисел в возрастающем порядке b1,b2,…,bm (−109≤b1<b2<⋯<bm≤109; bi≠0) — дома покупателей.



Сумма n по всем наборам входных данных не превосходит 2⋅105. Сумма m по всем наборам входных данных не превосходит 2⋅105.



Выходные данные

На каждый набор входных данных выведите одно целое число — максимальное количество коробок, которые можно оставить у домов покупателей.

Затем можно пойти на 6 налево и встать в позицию −1, сдвинув коробку в −2. В конце коробки стоят на позициях [−2,6,7,11,15], соответственно. Среди них позиции [−2,6,7,15] — дома покупателей, поэтому ответ равен 4.
